## Descripción de la Actividad

Esta actividad tuvo como objetivo comprender y aplicar técnicas de ingeniería social y phishing en un entorno estrictamente académico, controlado y supervisado.
El propósito fue analizar cómo funcionan las campañas maliciosas, identificar sus componentes técnicos y desarrollar criterios para detectarlas, mitigarlas y reportarlas.

No se persiguió ningún fin operativo fuera del ámbito educativo.

## Alcance y Consideraciones Éticas

Esta simulación de phishing se realizó exclusivamente con fines académicos, como parte de una actividad del curso.

Antes de su ejecución, se informó y contó con la autorización del docente encargado.

Para la prueba se usó una contraseña genérica (“1234”) y no se capturó información real de credenciales.

No se involucraron terceros, ni se envió la simulación fuera del entorno controlado.

Una vez completada la actividad, todo el código y los datos asociados fueron eliminados, manteniendo únicamente este README explicativo.
